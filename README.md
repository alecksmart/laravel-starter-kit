# laravel-starter-kit
Latest Laravel blog boilerplate starter kit application.

# Features

* Laravel
* MySQL migrations
* jQuery

# Pre-requisites

 * npm
 * composer

# Install

```bash
# preapre all
$ git clone https://github.com/alecksmart/laravel-starter-kit.git
$ cd laravel-starter-kit
$ composer install
$ npm install
$ php artisan migrate --seed
# or
$ php artisan migrate:refresh --seed
```
